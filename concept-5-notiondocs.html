Features

Powerful documentation, reimagined

BL

Block-based editor

63 block types including text, images, code, embeds, and databases. Drag and drop to reorganize.

DB

Linked databases

Create relational databases with multiple views: table, kanban, calendar, gallery, and timeline.

AI

AI writing assistant

Generate, summarize, translate, and improve your content with built-in AI powered by OpenAI.

RT

Real-time collaboration

See who's editing in real-time. Presence indicators, comments, and @mentions built in.

TM

Template gallery

100+ pre-built templates for wikis, project management, meeting notes, and more.

API

Full API access

REST and webhooks for integrations. Import/export in Markdown, HTML, and PDF.
